
1. Создайте класс Media
 - создайте поля (название, продолжительность, жанр)
 - создайте конструктор для всех полей
 - создайте метод display() - который распечатает все поля

2. Создайте класс Movie
 - создайте поле рейтинг
 - унаследуйте от класса Media
 - в конструкторе инициализируйте все поля
 - переопределите метод display() чтобы еще выводился рейтинг

3. Создайте класс Song
 - создайте поле исполнитель
 - унаследуйте от класса Media
 - в конструкторе инициализируйте все поля
 - переопределите метод display() чтобы еще выводился исполнитель

4. Создайте класс Series (сериал)
 - создайте поля колВоСерий
 - унаследуйте от класса Movie
 - в конструкторе инициализируйте все поля
 - переопределите метод display() чтобы еще выводился колВоСерий

5. Создайте класс с методом main
 - создайте объекты классов Movie, Song, Series
 - вызовите у объектов метод display()
